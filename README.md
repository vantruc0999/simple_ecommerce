# simple_ecommerce
This is a small ecommerce website using html, css, bootstrap, jquery, ajax as front-end and PHP as backend
_As an admin I can: 
  + Manage user
  + Manage category
  + Manage product 
  + Manage order
_As a user I can:
  + View product
  + Add product to cart
  + Manage cart 
  + Check out
  + View order history
